
加载完类之后，在堆内存的方法区中就产生了一个Class类型的对象（一个类只有一个Class对象），这个对象就包含了完整的类的结构信息。我们可以通过这个对象看到类的结构。这个对象就像一面镜子，透过这个镜子看到类的结构，所以，我们形象的称之为：反射。

* 作用：通过java中的反射机制可以操作字节码文件，可以操作代码片段（class）
* 反射机制在java.lang.reflect.\*
* 相关重要类:
+ java.lang.Class:代表字节码文件，代表整个类
+ java.lang.reflect.Method: 代表字节码中的方法字节码，代表方法
+ java.lang.reflect.Constructor: 代码字节码中的构造方法字节码
+ java.lang.reflect.Field: 代表字节码中的属性字节码，代表成员变量（静态变量+实例变量）

![](反射_001.png)jdk7之后存放在堆中

• Class 本身也是一个类

• Class 对象只能由系统建立对象

• 一个加载的类在 JVM 中只会有一个 Class 实例

• 一个 Class 对象对应的是一个加载到 JVM 中的一个 .class 文件

• 每个类的实例都会记得自己是由哪个 Class 实例所生成

• 通过 Class 可以完整地得到一个类中的所有被加载的结构

• Class 类是 Reflection 的根源，针对任何你想动态加载、运行的类，唯有先获得相应的Class 对象

```java
public class Demo01 {
    public static void main(String[] args) {

        //这是不使用反射机制，创建对象
        User user = new User();
        System.out.println(user);

        try {
            //通过反射机制，获取Class，来实例化对象
            Class c = Class.forName("class1.User");//c代表User类型
            //newInstance会调用User类无参构造方法创建对象，必须保证无参构造实现
            Object obj = c.newInstance();

            // class.newInstance() deprecated since JAVA9。使用无参构造方法来实例化对象
            c.getDeclaredContrustor().newInstance();
            System.out.println(obj);//class1.User@1b6d3586

            //如果构造器的权限修饰符修饰的范围不可见，也可以调用setAccessible(true)
        } catch (ClassNotFoundException e) {
            e.printStackTrace();
        } catch (IllegalAccessException e) {
            e.printStackTrace();
        } catch (InstantiationException e) {
            e.printStackTrace();
        }

    }
}
```

### 关于路径问题

上面的方式路径移植性差，在IDEA中默认当前路径是project的根，离开IDEA就无效了

使用以下方式的前提是，文件放在类路径下（凡是在src下的都是类路径）src是类的根路径

```java
public class AboutPath {
    /*
    Thread.currentThread() 当前线程对象
    getContextClassLoader()线程对象的方法，可以获得当前线程的类加载器对象
    getResource() 类加载器对象的方法，当前线程的类加载器默认从类的根路径下加载资源
     */
    public static void main(String[] args) {
        String path = Thread.currentThread()
                .getContextClassLoader()
                .getResource("classinfo.properties")
                .getPath();
        //以上代码可以获得一个文件的绝对路径，前提是文件在src路径下
        System.out.println(path);

        //以流的方式返回
        InputStream reader = Thread.currentThread().getContextClassLoader()
            .getResourceAsStream("classinfo.properties");
    }
}
```

## 获取Field

```java
public class ReflectTest05 {
    public static void main(String[] args) throws ClassNotFoundException {
        //获取整个类
        Class studentClass = Class.forName("class1.Student");
        System.out.println(studentClass.getName());//获得完整类名：class1.Student
        System.out.println(studentClass.getSimpleName());//简易类名：Student
        System.out.println("======================");

        //获取属性
        //这种方法只能拿到公共的属性名
        Field[] fields = studentClass.getFields();
        Field field = fields[0];
        System.out.println(field.getName());//no
        System.out.println("======================");

        //这种方法可以拿到所有属性名
        Field[] fs = studentClass.getDeclaredFields();
        for (Field f : fs) {
            //获取属性名
            System.out.print("属性名：" + f.getName() + "\t");//no name age sex
            //获取属性类型
            Class fieldType = f.getType();//返回类型为Class
            String fName = fieldType.getName();
            System.out.print("属性类型：" + fName + "\t");//int java.lang.String int boolean
            //获取修饰符列表
            int i = f.getModifiers();//返回的修饰符是一个数字，每个数字是修饰符的代号
            String modifierString = Modifier.toString(i);
            System.out.println("属性修饰符：" + modifierString + "\t");
        }

    }
}

class Student{
    //Field翻译为字段
    //4个Field四个访问权限
    public int no;
    private String name;
    protected int age;
    boolean sex;
}
```

```java
public class ReflectTest06 {
    public static void main(String[] args) throws ClassNotFoundException {
        //创建这个为了拼接字符串
        StringBuilder s = new StringBuilder();

        Class intClass = Class.forName("java.lang.Integer");

        s.append(Modifier.toString(intClass.getModifiers()) + " class" + intClass.getSimpleName() + "{\n");

        Field[] fields = intClass.getDeclaredFields();

        for (Field field : fields) {
            s.append("\t");
            s.append(Modifier.toString(field.getModifiers()));
            s.append(" ");
            s.append(field.getType().getSimpleName());
            s.append(" ");
            s.append(field.getName());
            s.append("\n");
        }

        s.append("}");
        System.out.println(s);
    }
}
```

### 反射机制访问对象属性

```java
/*
怎么通过反射机制访问一个对象的属性
	给属性赋值set
    获取属性的值get
*/
public class ReflectTest07 {
    public static void main(String[] args) throws Exception {
        //不使用反射机制
        Student s = new Student();
        s.no = 1111;

        //使用反射机制怎么访问对象属性
        Class studentClass = Class.forName("class1.Student");
        Object obj = studentClass.newInstance();//obj是Student对象

        //获取no属性（根据属性名来获取Field）
        Field noField = studentClass.getDeclaredField("no");

        //给obj对象(Student对象)的no属性赋值
        noField.set(obj,2222);

        //读取属性的值
        System.out.println(noField.get(obj));

        //可以访问私有属性吗
        Field nameField = studentClass.getDeclaredField("name");
        //打破封装(缺点：打破封装有安全隐患，外部也可以访问private)
        nameField.setAccessible(true);
        //给name属性赋值
        nameField.set(obj,"jackson");
        //获取name属性值
        System.out.println(nameField.get(obj));
    }
}
```

```java
public class UserService {
    /**
     *
     * @param name 用户名
     * @param password 密码
     * @return true表示成功 false表示失败
     */
    public boolean login(String name, String password){
        if ("admin".equals(name) && "123".equals(password)){
            return true;
        }
        return false;
    }

    public void logout(){
        System.out.println("已退出登陆");
    }

public class ReflectTest08 {
    public static void main(String[] args) throws Exception{
        //获取类
        Class userServiceClass = Class.forName("class1.UserService");

        //获取所有Method
        Method[] methods = userServiceClass.getDeclaredMethods();

        //遍历Method
        for (Method method : methods) {
            //获取修饰符列表
            System.out.print(Modifier.toString(method.getModifiers()) + " ");
            //获取方法返回值类型
            System.out.print(method.getReturnType().getSimpleName() + " ");
            //获取方法名
            System.out.print(method.getName());
            System.out.print("(");
            //方法的参数列表
            Class[] parameterTypes = method.getParameterTypes();
            for (Class parameterType : parameterTypes) {
                System.out.print(parameterType.getSimpleName());
                System.out.print(",");
            }
            System.out.println(")");

        }
    }
}
```

### 反射机制调用方法

```java
/*
通过反射机制如何调用一个对象的方法
 */
public class ReflectTest10 {
    public static void main(String[] args) throws Exception {
        //不使用反射机制，怎么调用方法
        UserService userService = new UserService();
        boolean loginSuccess = userService.login("admin","123");
        System.out.println(loginSuccess ? "登陆成功" : "登陆失败");

        //使用反射机制来调用一个对象的方法应该怎么做
        Class userServiceClass = Class.forName("class1.UserService");
        //创建对象
        Object obj = userServiceClass.newInstance();
        //获取Method(注意区别重载的方法)
        Method loginMethod = userServiceClass.getDeclaredMethod("login", String.class, String.class);
        //调用方法 ***重要！***
        Object retValue = loginMethod.invoke(obj,"admin","321");
        System.out.println((Boolean)retValue ? "登陆成功" : "登陆失败");
    }
}
```

### 反射机制还原构造方法

```java
public class ReflectTest11 {
    public static void main(String[] args) throws Exception{
        StringBuilder s = new StringBuilder();
        Class vipClass = Class.forName("class1.Vip");
        s.append(Modifier.toString(vipClass.getModifiers()));
        s.append(" class ");
        s.append(vipClass.getSimpleName());
        s.append("{\n");

        //拼接构造方法
        Constructor[] constructors = vipClass.getDeclaredConstructors();
        for (Constructor constructor : constructors) {
            s.append("\t");
            s.append(Modifier.toString(constructor.getModifiers()));
            s.append(" ");
            s.append(vipClass.getSimpleName());
            s.append("(");
            //拼接参数
            Class[] parameterTypes = constructor.getParameterTypes();
            for (Class parameterType : parameterTypes){
                s.append(parameterType.getSimpleName());
                s.append(",");
            }
            if (parameterTypes.length > 0) {
                s.deleteCharAt(s.length() - 1);//删除最后一个逗号
            }
            s.append("){}\n");
        }
        s.append("}");
        System.out.println(s);
    }
}
```

```java
public class ReflectTest13 {
    public static void main(String[] args) throws ClassNotFoundException {
        //String举例
        Class stringClass = Class.forName("java.lang.String");

        //获取String父类

            java.lang.Comparable
            java.lang.CharSequence

        }
    }
}
```
