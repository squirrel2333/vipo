# 标准数据类型

Python3 中有六个标准的数据类型：

* Number（数字）
* String（字符串）
* List（列表）
* Tuple（元组）
* Set（集合）
* Dictionary（字典）

Python3 的六个标准数据类型中：

* **不可变数据（3 个）：**Number（数字）、String（字符串）、Tuple（元组）；
* **可变数据（3 个）：**List（列表）、Dictionary（字典）、Set（集合）。

**可变类型: 内存地址不变, 内容改变**

**不可变类型: 变量对应的值不能修改, 如果修改就会生成新的内存空间**

# 类型

## 类型转换

* `str()`将其他类型转为str，也可以用""转换
* `int()`转为整数类型，浮点数=>截取整数部分；整数字符串 =>int类型（非整数字符串无法转换）；布尔类型=>1或0；
* `float()`转换为浮点类型。文字无法转为浮点；整数字符串后面加.0；布尔1.0

```python
name = '张三'
age = 20
print('My name is ' + name + ' ,I am ' + str(age) + 'years old')  # 必须要类型转换，才能连接
```

```python
# 整数转字符串
num_str = str(11)
print(type(num_str), num_str) # <class 'str'> 11

# 浮点转字符串
float_str = str(11.345)
print(type(float_str), float_str)

# 字符串转整数
num = int("11")

# 字符串转浮点
num2 = float("11.233")

# 转换必须满足要求
print(int("hello"))
# ValueError: invalid literal for int() with base 10: 'hello'

# 浮点转整数 -》 丢失精度
int_num = int(11.2) # 11
```
* `int()`：将一个数值或字符串转换成整数，可以指定进制。
* `float()`：将一个字符串（在可能的情况下）转换成浮点数。
* `str()`：将指定的对象转换成字符串形式，可以指定编码方式。
* `chr()`：将整数（字符编码）转换成对应的（一个字符的）字符串。
* `ord()`：将（一个字符的）字符串转换成对应的整数（字符编码）。

```python
print(int(c, base=16))  # str类型的'123'按十六进制转成int，输出291
print(int(d, base=2))   # str类型的'100'按二进制转成int，输出4
```
## 类型判断

isinstance 和 type 的区别在于：

* type()不会认为子类是一种父类类型。
* isinstance()会认为子类是一种父类类型。

```python
>>> class A:
...     pass
...
>>> class B(A):
...     pass
...
>>> isinstance(A(), A)
True
>>> type(A()) == A
True
>>> isinstance(B(), A)
True
>>> type(B()) == A
False
```
注意：Python3 中，bool 是 int 的子类，True 和 False 可以和数字相加， True==1、False==0 会返回 True，但可以通过 is 来判断类型。

```python
>>> issubclass(bool, int)
True
>>> True==1
True
>>> False==0
True
>>> True+1
2
>>> False+1
1
>>> 1 is True
False
>>> 0 is False
False
```
*在 Python2 中是没有布尔型的，它用数字 0 表示 False，用 1 表示 True。*
