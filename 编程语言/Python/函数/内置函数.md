# abs

返回绝对值

# sum

函数内要放可迭代对象

字符串、整型、字典都不行

列表，元组，集合可以

```python
sum({1,5,3})
```
# min

```python
min(4,1,8)
min(-8,5,key=abs)) #绝对值比较
```
# max

```python
max([4,1,6])
```
# zip

将可迭代对象作为参数，将对象中对应的元素打包成元组

```python
li = [1, 2, 3]
li2 = ['a','b','c']
for i in zip(li, li2):
    print(i) # (1,'a') (2,'b') (3,'c')
```
