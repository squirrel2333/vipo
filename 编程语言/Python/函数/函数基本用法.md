# 函数变量

* 函数内部使用变量，会先从函数内部找，找不到就会从函数外部找。

```python
num = 200

def test():
    num = 500

test()
print(num) # 200 全局变量不会被内层的修改
```
* 在函数内可以通过声明`global num`将`num`声明为全局变量， 也可以对全局变量进行修改

```python
tip = """
xxx, 您好，欢迎来到银行ATM。请选择操作：
查询余额  [输入1]
存款     [输入2]
取款     [输入3]
退出     [输入4]
请输入您的选择：
"""

money = 5000000

def save_money():
    print("---------------存款--------------")
    save_money = int(input("存款金额："))
    global money
    money = money + save_money

def withdraw_money():
    print("---------------存款--------------")
    save_money = int(input("存款金额："))
    global money
    money = money - save_money

def search_money():
    print("----------------查询余额----------------")
    print(f"xxx，您好，您的余额为：{money}")

def main():

    print(tip)
    input_num = int(input())
    if input_num == 1:
        search_money()
    elif input_num == 2:
        save_money()
        search_money()
    elif input_num == 3:
        withdraw_money()
        search_money()

while True:
    main()
```
* nolocal: 用来声明外层的局部变量，只能在前提函数中使用

```python
a = 1
def outer():
    a = 2
    def inner():
        nonlocal a
        a = 3
        print("inner a ", a)
    inner()
    print("outer a", a)
outer()
# inner a 3
# outer a 3
```

# 值传递与引用传递

在Python中，函数的参数传递是“对象引用的传递”。这并不是传统意义上的“值传递”或“引用传递”，而是一个更接近于“引用传递”的机制。以下是具体的解释：

**工作原理：**

当一个值作为参数传递给函数时，函数接收到的是这个值的引用。

1. **可变对象引用**：

如果参数是一个可变对象（如列表、字典等），函数内对该对象的修改会影响到原对象。

```python
def modify_list(lst):
    lst.append(4)

my_list = [1, 2, 3]
modify_list(my_list)
print(my_list)  # 输出: [1, 2, 3, 4]
```
2. **可变对象**：

如果参数是一个不可变对象（如整数、字符串、元组等），函数内的修改不会影响到原对象，因为不可变对象的操作实际上会创建一个新的对象。

```python
def modify_number(num):
    num += 10

my_num = 5
modify_number(my_num)
print(my_num)  # 输出: 5
```
