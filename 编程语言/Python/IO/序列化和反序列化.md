# Json处理

|  |  |
| --- | --- |
| **Python** | **JSON** |
| `dict` | `object` |
| `list`/ `tuple` | `array` |
| `str` | `string` |
| `int` / `float` | `number` |
| `bool`（`True`/ `False`） | `boolean`(`true`/ `false`) |
| `None` | `null` |

`json`模块有四个比较重要的函数，分别是：

* `dump` - 对象->文件（将Python对象按照JSON格式序列化到文件中）
* `dumps` - 对象->字符串（将Python对象处理成JSON格式的字符串）
* `load` - 文件->对象 （将文件中的JSON数据反序列化成对象）
* `loads` - 字符串->对象（将字符串的内容反序列化成Python对象）

## dumps

```python
import json

my_dict = {
    'name': '骆昊',
    'age': 40,
    'friends': ['王大锤', '白元芳'],
    'cars': [
        {'brand': 'BMW', 'max_speed': 240},
        {'brand': 'Audi', 'max_speed': 280},
        {'brand': 'Benz', 'max_speed': 280}
    ]
}
print(json.dumps(my_dict))
```
运行上面的代码，输出如下所示，可以注意到中文字符都是用Unicode编码显示的。

{"name": "\u9a86\u660a", "age": 40, "friends": ["\u738b\u5927\u9524", "\u767d\u5143\u82b3"], "cars": [{"brand": "BMW", "max\_speed": 240}, {"brand": "Audi", "max\_speed": 280}, {"brand": "Benz", "max\_speed": 280}]}

如果要将字典处理成JSON格式并写入文本文件，只需要将`dumps`函数换成`dump`函数并传入文件对象即可，代码如下所示。

## dump

```python
import json

my_dict = {
    'name': '骆昊',
    'age': 40,
    'friends': ['王大锤', '白元芳'],
    'cars': [
        {'brand': 'BMW', 'max_speed': 240},
        {'brand': 'Audi', 'max_speed': 280},
        {'brand': 'Benz', 'max_speed': 280}
    ]
}
with open('data.json', 'w') as file:
    json.dump(my_dict, file)
```
## load

```python
import json

with open('data.json', 'r') as file:
    my_dict = json.load(file)
    print(type(my_dict))
    print(my_dict)
```
## loads

```python
import json
# 字典转json
data = [{"name":"zhangsan", "age": 11}, {"name": "wangwu", "age": 12}]

json_str = json.dumps(data, ensure_ascii=False)
print(type(json_str))  #<class 'str'>
print(json_str)

# 字符串转python类型
s = '[{"name":"zhangsan", "age": 11}, {"name": "wangwu", "age": 12}]'
l = json.loads(s)
print(type(l))  # <class 'list'>
print(l)

s = '{"name":"zhangsan", "age": 11}'
d = json.loads(s)
print(type(d)) # <class 'dict'>
print(d)
```
