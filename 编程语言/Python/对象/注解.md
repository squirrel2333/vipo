# 类型注解

类型注解是提示性的,而不是决定性的. **并不会真正对类型做判断**, 同样可以执行, 只是方便IDE进行代码推断

## 变量类型注解

* 基础类型注解
* 对象注解

```python
var1: int = 10

my_list: list = [1, 2, 3]

# 详细注解容器内部的类型
my_list: list[int] = [1, 2, 3]
# tuple必须每个元素都标记
my_tuple: tuple[str, int, bool] = ("itheima", 555, True)
# dict必须注解两个类型k,v
my_dict: dict[str, int] = {"hhh": 23}

stu: Student = Student()
```
也可以通过注释进行注解

```python
val_1 = random.randint(1, 10) # type: int
```
## 函数类型注解

```python
def func(x: int, y: int) -> int:
    return x + y

def func2(data: list) -> list:
    return data
```
## Union类型

混合类型, 可以是多种类型

`Union[x,x,....]`

```python
form typing import Union

my_list: list[Union[str, int]] = [1, 2, "ith", "ikk"]
my_dict: dict[str, Union[str, int]] ={"name": "zzz", "age": 23}

def func(data: Union[int, str]) -> Union[int, str]:
```
