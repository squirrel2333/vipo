模块就是一个python文件, 模块定义函数, 类, 和变量, 模块里也可能包含可执行的代码

模块的作用: 把不同功能封装在一起

# 模块导入

`[from 模块] import [模块|类|变量|函数|\*] [as 别名]`

```python
import http
from time import sleep
from time import *
import time as t
from html.parser import HTMLParser
```
# main函数

当在模块中需要执行命令, 但不想被其它包导入时就直接执行, 于是定义main方法

当点运行时, python会把\_\_name\_\_变量变成字符串, if就能进入

```python
if __name__ = '__main__':
```
# 自定义模块

```python
from my_module1 import test
from my_module2 import test

# 用的是module2的test, 后导入的被覆盖
test(1, 2)

# 如果模块中有__all__变量, 当使用from xxx import * 时, 只能导入列表中的元素
__all__ = ['test_a']

def test_a():

def test_b():

```
