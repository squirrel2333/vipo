# 安装启动

## 下载安装

```
# wget http://download.redis.io/releases/redis-7.0.2.tar.gz
# tar xzf redis-7.0.2.tar.gz
```
## 安装

```
cd redis-6.0.8

make #make他会帮你安装gcc及其他环境 从Makefile中读取指令编译

make install #安装到指定位置 也可以直接make && make install
```
默认安装路径`/usr/local/bin`

redis-benchmark 性能测试工具

redis-check-aof -> redis-server 修复有问题的AOF文件

redis-check-rdb -> redis-server 修复有问题的rdb文件

redis-cli 客户端

redis-sentinel -> redis-server 集群

redis-server 服务端

## 配置

```
# cd src
# ./redis-server ../redis.conf

redis-server /path/to/your/redis.conf --daemonize yes#后台启动
```
可以将redis.conf复制一份修改，并在启动时加载

```
daemonize no #服务器默认为前台启动，改为yes后台启动
protected-mode yes #保护模式，改为no允许其他host访问
bind 127.0.0.1 #注释掉，即可允许远程访问
requirepass 123456 #取消注释，即可设置密码
```
## 启动

1. 执行完 **make** 命令后，redis-6.0.8 的 **src** 目录下会出现编译后的 redis 服务程序 redis-server，还有用于测试的客户端程序 redis-cli：

```
# cd src
# ./redis-server
```

```
# cd src
# ./redis-cli
redis> set foo bar
OK
redis> get foo
"bar"
```
2. **或者，因为bin中包含了启动命令所以可以直接使用**

`[root@VM-4-13-centos] redis-server ~/redis-7.2.0/redis1.conf`

`[root@VM-4-13-centos redis-7.2.0]# ps -ef|grep redis`

`root 2920 1 0 13:36 ? 00:00:00 redis-server \*:6379`

## 连接服务

`redis-cli -a Hqw3.1415926 -p 6379`

# 关闭

## 断开服务

`quit`

## 关闭server

1. 本地环境直接杀进程

`ps -aux|grep redis`

`kill -9 17899`

2. 或者在cli中`SHUTDOWN`
3. 或者命令`redis-cli -a Hqw31.141 shutdown`
4. 指定端口关闭`redis-cli -p 6379 shutdown`

# 卸载

1. 停止服务
2. 删除/usr/local/bin/redis\*
