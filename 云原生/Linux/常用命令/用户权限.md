# 重置密码

`sudo passwd <user>`

# 给用户root权限

第一步，新增用户：

　　adduser test

第二步，修改密码：

　　passwd test

第三步，赋管理员权限：

　　1）先修改配置文件为可编辑：chmod -v u+w /etc/sudoers

　　2）修改：vim /etc/sudoers

　　3）赋权：找到root ALL=(ALL) ALL 在下边加一行 test ALL=(ALL) NOPASSWD :ALL

　　4）说明：NOPASSWD :ALL代表的含义是在test用户进行sudo操作指令的时候不需要输入密码

第四步，将权限管理的配置文件设置为只读：

　　chmod -v u-w /etc/sudoers
