# Java

1. update/usr/bin/下多存放用户的安装软件 /usr/lib/下存放系统安装软件

`ll /usr/bin/java`

`lrwxrwxrwx 1 root root 22 Mar 13 22:31 **java** -> /etc/alternatives/**java**`

2. alternatives是linux多版本管理软件，利用它可以实现jdk切换

查看响应的jdk是否在菜单里，选择序号

```
[root@izuf65mhum14tjzqrobqrsz alternatives]# update-alternatives --config java

There are 2 programs which provide 'java'.

  Selection    Command
-----------------------------------------------
*+ 1           /usr/java/jdk1.8.0_291-amd64/bin/java
   2           java-11-openjdk.x86_64 (/usr/lib/jvm/java-11-openjdk-11.0.14.1.1-1.el7_9.x86_64/bin/java)
```
如果没有可以如下添加：`update-alternatives --install /usr/bin/java java /usr/lib/jvm/java/jdk1.6.0\_12/bin/java 300`

3. `java -version`查看版本

# Maven

```
wget https://archive.apache.org/dist/maven/maven-3/3.3.9/binaries/apache-maven-3.3.9-bin.tar.gz
tar xzf apache-maven-3.3.9-bin.tar.gz
mkdir /usr/local/maven
mv apache-maven-3.3.9/ /usr/local/maven/
update-alternatives --install /usr/bin/mvn mvn /usr/local/maven/apache-maven-3.3.9/bin/mvn 1
update-alternatives --config mvn
```
