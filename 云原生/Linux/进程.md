1. **使用** **ps** **命令查看进程**：

```
perlCopy code
ps aux | grep <进程名或关键词>
```
这将列出所有包含指定关键词的进程，以及它们的 PID（进程ID）。

2. **使用** **kill** **命令杀死进程**：

```
bashCopy code
kill <PID>
```
或者如果进程无法通过正常方式结束，你可以使用 **kill -9** 命令：

```
bashCopy code
kill -9 <PID>
```
这会强制结束进程。

举例来说，如果要杀死名为 **myprocess** 的进程，你可以这样做：

```
perlCopy code
ps aux | grep myprocess
```
这会显示包含 **myprocess** 的进程以及它们的 PID。然后，选择你想要终止的 PID，并使用 **kill** 命令终止它：

```
bashCopy code
kill <PID>
```
或者使用 **kill -9**：

```
bashCopy code
kill -9 <PID>
```
请记住，使用 **kill -9** 可能会强制结束进程并且可能导致数据损坏或其他意外情况，所以最好先尝试使用 **kill** 命令，只有在必要时才使用 **kill -9**。
