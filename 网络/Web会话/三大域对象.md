
域对象：用于存储和传递数据对象。传递数据不同的范围，称为不同的域。不同域对象代表不同的域，共享数据的范围也不同

* web项目中，域对象分别为：请求域、会话域、应用域
+ 请求域对象：HttpServletRequest。传递数据范围是一次请求之内及请求转发
+ 会话域对象：HttpSession。传递数据范围是一次会话之内，可以跨多个请求
+ 应用域对象：ServletContext，传输数据的范围是本应用之内，可以跨多个会话

# 域对象的使用

`void setAttribute(String name, String value)`向域对象中添加/修改数据

`Objet getAttribute(String name)`域对象中获取数据

`void removeAttribute(String name)`移除域对象中的数据

## 放入数据

```
@WebServlet("/servletA")
public void ServletA extends HttpServlet {
    @Override
    protected void service(HttpServletRequest req, HttpServletResponse resp) throws ServletException,IOException {
        //向请求域中放入数据
        req.setAttribute("request", "request-message");
        //req.getRequestDispatcher("servletB").foward(req.resp);
        //向会话域中放入数据
        HttpSession session = req.getSession();
        session.setAttribute("session", "session-message");
        //向应用域中放入数据
        ServletContext application = this.getServletContext();
        application.setAttribute("application", "application-message");
    }
}

```
## 取出数据

```
@WebServlet("/servletB")
public class ServletB extends HttpServlet {
    @Override
    protected void service(HttpServletRequest req, HttpServletResponse resp) throws SevletException,IOException {
        // 从请求域中获取数据
        String reqMessage = (String)req.getAttribute("request");
        System.out.println(reqMessage);
        //从会话域中
        String sessionMessage = (String)req.getSession().getAttribute("session");
        // 从应用域中
        ServletContext application = getServletContext();
        String applicationMessage = (String)application.getAttribute("application");
    }
}
```
